### VEP - Vietnam Education Platform

VEP is an AI-powered education platform designed to support online learning, document management, exam practice, and automatic question generation from learning materials.

**Tech Stack:**  
Next.js, React, TypeScript, Tailwind CSS, Node.js, Express.js, Prisma, PostgreSQL, JWT, Google Gemini API

**AI / LLM Integration:**  
Google Gemini API, LLM Prompt Design, AI Question Generation, PDF-based Content Processing, AI Writing Evaluation, Structured JSON Response Handling

**Key Features:**

- Built a responsive dashboard for students, teachers, and administrators
- Developed REST APIs for authentication, documents, subjects, practice tests, and APTIS modules
- Implemented role-based access control using JWT authentication
- Designed PostgreSQL database schema with Prisma ORM
- Integrated Google Gemini API to generate multiple-choice questions from PDF content
- Built APTIS practice modules including Writing, Reading, Listening, and Speaking
- Developed learning features such as leaderboard, discussion forum, document approval, and AI writing evaluation
